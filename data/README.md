# Data

This folder contains the instance data used in the experiments.

## Contents

- **all_file.rar** — Benchmark instances (extract before use).
- **Instances Detail.pdf** — Description of the instances (source, format, and details of each instance used in the paper).

The instances are from the public libraries QPLIB and MINLPLIB. Sources, licenses, and citations are given below.
